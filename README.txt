GeopIP is a class for matching IP's to Countries.
On starup it loads every range into the memory and when looking up an IP it binary searches through this ranges.


Classes:
    - Ip: class representing an IP
    - Range: class representing an IP range
    - GeoIP: class for matching IP's to Countries
    
Main:
    Just a script to show off...
